# 边缘推理与嵌入式 AI — 文献调研

## 1. TensorFlow Lite / TFLite

### TensorFlow Lite Micro: Embedded Machine Learning on TinyML Systems
- **作者**：David, R., Duke, J., Jain, A., Reddi, V. J., et al.
- **年份**：2021（arXiv 2020）
- **出版**：MLSys 2021
- **链接**：https://arxiv.org/abs/2010.08678 | https://proceedings.mlsys.org/paper_files/paper/2021/file/6c44dc73014d66ba49b28d483a8f8b0d-Paper.pdf
- **摘要**：专为资源极度受限的嵌入式 MCU 设计的 TFLite Micro 框架，在仅数 KB 内存的处理器上运行神经网络推理，是 TFLite 在嵌入式端的核心参考论文。

## 2. ONNX Runtime

### Accelerating Deep Learning Model Inference on ARM-based Devices via ONNX Runtime
- **作者**：ARM / 微软研究团队
- **年份**：2023–2024
- **出版**：arXiv 2207.08820
- **链接**：https://arxiv.org/pdf/2207.08820 | https://learn.arm.com/learning-paths/servers-and-cloud-computing/onnx-on-azure/benchmarking/
- **摘要**：在 Azure Cobalt 100（ARM64）及边缘 ARM 设备上优化 ONNX Runtime 推理，利用 KleidiAI 算子库实现 28–51% 性能提升。

### A Selective Quantization Tuner for ONNX Models (SeQTO)
- **作者**：Louloudakis, N., et al.
- **年份**：2025
- **出版**：arXiv 2507.12196
- **链接**：https://arxiv.org/abs/2507.12196
- **摘要**：对 ONNX 模型进行选择性量化部署的自动调优框架，在 Intel 和 Mali GPU 上分别实现 72.9% 精度提升和 54.14% 模型体积压缩。

### Pre-Quantized Deep Learning Models Codified in ONNX to Enable Hardware/Software Co-Design
- **作者**：研究团队
- **年份**：2021
- **出版**：arXiv 2110.01730
- **链接**：https://arxiv.org/abs/2110.01730
- **摘要**：提出以 ONNX 格式存储预量化模型，将量化过程与硬件编译解耦，支持跨平台嵌入式部署。

## 3. 树莓派部署现状

### Benchmarking Deep Learning Models for Object Detection on Edge Computing Devices
- **作者**：研究团队
- **年份**：2024
- **出版**：arXiv 2409.16808
- **链接**：https://arxiv.org/html/2409.16808v1
- **摘要**：对 YOLOv8、EfficientDet Lite、SSD 等主流目标检测模型在树莓派多版本（RPi 4/5）上进行推理性能基准测试，量化延迟、吞吐量和功耗。

### Exploring Distributed Deep Learning Inference Using Raspberry Pi Spark Cluster
- **作者**：研究团队
- **年份**：2022
- **出版**：Future Internet, Vol. 14, Issue 8（MDPI）
- **链接**：https://www.mdpi.com/1999-5903/14/8/220
- **摘要**：利用树莓派集群和 Apache Spark 进行分布式深度学习推理的可行性研究与性能评估。

## 4. 量化方法综述（与嵌入式推理相关）

### A Survey of Quantization Methods for Efficient Neural Network Inference
- **作者**：Gholami, A., Kim, S., Dong, Z., et al.
- **年份**：2021
- **出版**：arXiv 2103.13630
- **链接**：https://arxiv.org/abs/2103.13630
- **摘要**：系统综述 PTQ、QAT、低比特量化、混合精度等方法，CPU/GPU 上实现 2–10× 推理加速，精度损失 < 2%。

### Quantizing Deep Convolutional Networks for Efficient Inference: A Whitepaper
- **作者**：Jacob, B., et al.（Google AI）
- **年份**：2018
- **出版**：arXiv 1806.08342
- **链接**：https://arxiv.org/abs/1806.08342
- **摘要**：Google 官方 8bit 量化指南，介绍后训练量化（PTQ）在多种 CNN 架构上的实现方式，精度损失 < 2%，推理加速 2–10×。
